# Environment Variables Reference

This document lists all environment variables used by BloomX and its expansions.

## Core

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | PostgreSQL connection string. |
| `NEXT_PUBLIC_APP_URL` | The public URL of the application (e.g., `https://my-bloomx.com`). |
| `COOKIE_SECRET` | Secret key for signing session cookies (min 32 chars). |
| `EXPANSION_SECRET` | Secret key for encrypting stored expansion credentials (min 32 chars). |
| `RESEND_API_KEY` | API Key for Resend email service. |
| `REGISTRATION_KEY` | Secret code required for new users to sign up. |

## AI Providers

Configure at least one provider.

| Variable | Description |
|----------|-------------|
| `AI_PROVIDER` | `openai`, `gemini`, `anthropic`, or `cohere`. |
| `AI_OPENAI_API_KEY` | Key for OpenAI. |
| `AI_GEMINI_API_KEY` | Key for Google Gemini. |
| `AI_ANTHROPIC_API_KEY` | Key for Anthropic Claude. |
| `AI_COHERE_API_KEY` | Key for Cohere. |

## Storage (S3 Compatible)

| Variable | Description | required |
|----------|-------------|----------|
| `S3_ENDPOINT` | Endpoint URL (e.g. AWS, Cloudflare R2). | Yes |
| `S3_REGION` | Region (e.g. `us-east-1` or `auto`). | Yes |
| `S3_ACCESS_KEY` | Access Key ID. | Yes |
| `S3_SECRET_KEY` | Secret Access Key. | Yes |
| `S3_BUCKET` | Bucket name. | Yes |

## OAuth Integrations (Expansions)

To enable "Connect" buttons for these services, provide the Client ID and Secrets.

### Google (Drive, Meet, Gmail)
- `GOOGLE_CLIENT_ID`
- `GOOGLE_CLIENT_SECRET`

### Zoom (Meetings)
- `ZOOM_CLIENT_ID`
- `ZOOM_CLIENT_SECRET`

### Slack (Notifications, Commands)
- `SLACK_CLIENT_ID`
- `SLACK_CLIENT_SECRET`

### Notion (Pages, Databases)
- `NOTION_CLIENT_ID`
- `NOTION_CLIENT_SECRET`

### HubSpot (CRM)
- `HUBSPOT_CLIENT_ID`
- `HUBSPOT_CLIENT_SECRET`

### Trello (Tasks)
- `TRELLO_API_KEY` (Application Key)

### Generic CRM (Expansion)
- `EXPANSION_CRM_URL`
- `EXPANSION_CRM_API_KEY`
